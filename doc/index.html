Preface

Hotbabe Chess is a chess game written using Eiffel (Eiffel Studio 5.3), and designed to run under Windows.

This document offers a high level description of the source code, starting with the major clusters, and then providing class diagrams. There are also detailed instructions for compiling this application.

Introduction:

Hotbabe Chess is a windows chess program that includes a "hotbabe" character, and simulates an internet chat room.

History:

Being out of work and looking for some project to keep my Eiffel skills fresh, I decided in May of 2003 to submit a project to the Eiffel Struggle 2003 competition.

But what project?

At the time I was playing chess with my ex-gf over the internet. And I was starting to enjoy chess, and I started playing against people on the internet. I kept losing, so I used my Igor Chess game to help give me hints and play against people on the internet.

Eventually it struck me that I could write a chess engine, and that it would be a perfect project for the Eiffel Struggle 2003 competition, because:

  • Will give me more eiffel experience.
  • Is a unique project in the eiffel open source world
  • May lead to consulting work
  • Can be used to learn the new ISE Eiffel Development environment
  • Will allow me to fill out my resume/portfolio

At first my goal was a simple chess engine, with no GUI. I originally planned to interface with WinBoard/XBoard programs. But I eventually decided to make a complete game in Eiffel.

On April 13th, 2003 the idea for Hotbabe came to me. I was finished with the chess engine by this time, and was trying to think of really unique and intuitive user interfaces. Thats when I decided that a simulated internet chat room would make a great interface.

  1. Most users already understand this type of game
  2. Instructions can be displayed inside of the chat window
  3. Playing computer chess is more fun when the computer has some "character".
